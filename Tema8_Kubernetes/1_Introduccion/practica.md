# Práctica: Introducción a Kubernetes

1. Investiga qué empresas y proyectos usan Kubernetes actualmente.
2. Enumera las ventajas principales de usar Kubernetes frente a Docker Compose.
3. Instala Minikube o Kind en tu equipo para crear un cluster local de Kubernetes.
4. Comprueba el estado del cluster con:
   ```bash
   kubectl cluster-info
   kubectl get nodes
   ```
5. Reflexiona: ¿En qué casos usarías Kubernetes y en cuáles no?
